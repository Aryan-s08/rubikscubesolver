Each of the files contains solvers for the specified type of rubik's cube. They not only solve the rubik's cube but also give the sequence of blindfold letters which is extremely useful for those who are trying to learn
solving a cube blindfolded. Each letter corresponds to a small sequence of moves which with enough practice can be learnt and the cube can be solved by anyone with just the letters sequence. One can use these programs 
to check if they memorized the sequence correctly or not.
